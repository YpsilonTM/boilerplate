# boilerplate

## Project setup

Clone into resource folder

```
git clone git@github.com:YpsilonTM/boilerplate.git
```

Run NPM install to get all dependencies

```
npm install
```

## Package

So what is in the boilerplate:

- Vue 2
- Vuex
- Vuetify
- Axios
- Custom NUI Wrapper
- Empty LUA structure
